## 1.0.1

- `--filter` option: apply artistic filters from the command line.
- `--thumbnail` option: generate thumbnails preserving aspect ratio.
- `filters` command: list all available artistic filter names.
- `blurhash` command: encode images to BlurHash strings and decode back to PNG.

## 1.0.0

- Initial release.
- `process` command: resize, format convert, effects, watermarks.
- `info` command: display image dimensions, format, and file size.
